I separated scene loading into Scene.hpp where I use sejp to parse the scene, creating vectors of each resource and linking the resources together using their indices in the vector. Everything within the scene is contained in the scene object which is passed into the application. Frustum culling is seperated into frustum_culling.hpp, and headless event loading is seperated into HeadlessEvent.hpp which parses the .events file and returns a vector of HeadlessEvents. There isn't anything too fancy in my implementation.
